# Healthcare Workforce Housing Platform

Planning repository for the Healthcare Workforce Housing Platform.

## Purpose

This repository organizes the opportunity to create sustainable rental housing near major hospitals, beginning with the Sapperton Healthcare Housing District near Royal Columbian Hospital in New Westminster, British Columbia.

## Core Modules

1. Hospital Demand Module
2. Land Assembly Module
3. Program Readiness Module
4. Green Rental Housing Module
5. Modular Delivery Module
6. Partner Tracker
7. Reporting Module
